# Pop Menu Project

An application to manage restaurant menus.

## Project Structure

- `backend/` → Ruby on Rails API
- `frontend/` → React frontend
- `README.md` → this file

## Prerequisites

- Ruby 3.3.9 (via Homebrew)
- Node.js and npm
- PostgreSQL

## Running the Backend

1. Go to the backend folder:

```
cd backend
```

2. Install dependencies:

```
bundle install
```
